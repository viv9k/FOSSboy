Google Summer Of Code - 2019

Google Summer of Code is Global program that helps student to connect with Open Source, free software and technology related organization to write code and work on their open projects to make the internet a better place. Students need to submit their project proposal and work on the project for 3 months. In this program, I have submitted a proposal for making an agent into tool Fossology to integrate Microsoft's Clearly Defined into it.

My project was to an agent into fossology to integrate Clearly Defined API. Clearly Defined are library archive of copyright and license for Open Source Tools and most important they helps in collecting these information to reduce the lack of license and informations.
